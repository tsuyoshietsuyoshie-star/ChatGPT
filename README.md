# ChatGPT Desktop (Electron)

A lightweight, minimal and fast desktop wrapper for **ChatGPT**, built with **Electron**.  
This project allows you to run ChatGPT like a native Windows application, with optional mini-chat windows, tray integration, theme switching and UI injection.

This repository is ideal for developers who want to learn how to build browser-based desktop apps with Electron, or who want to reuse this structure for their own AI or web-based tools.
